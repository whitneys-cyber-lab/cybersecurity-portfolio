# SOC Investigations

This section contains hands-on security operations and incident investigation write-ups completed as part of my cybersecurity training and portfolio development.

These projects focus on analyzing logs, identifying suspicious activity, reconstructing timelines, documenting findings, and recommending defensive actions.

## Focus Areas

- SIEM investigation
- Log analysis
- Suspicious authentication activity
- Cloud security event review
- Incident timeline reconstruction
- Threat actor behavior analysis
- Analyst-style documentation
